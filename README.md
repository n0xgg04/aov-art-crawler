# AOV ART CRAWLER

- Fast AOV Crawler, save image automatically
### Installations:
Add your art API to ``constants/api.go``

```go
ArtAPI = map[string]string{
"cn": "",
"tw": "",
"vn": "",
}
```

Install go modules:
```bash
go get
```

### Run crawler
```bash
go run main.go
```

OR

```bash
npm run dev
```
